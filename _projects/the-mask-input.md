# How to Install?

```shell
npm install the-mask-input
```

# How can I use it?

You can import the component `<Input />` from `the-mask-input` and start to use

```typescript jsx
import Input from "the-mask-input";

<Input mask="cpf" placeholder="CPF" />
```
